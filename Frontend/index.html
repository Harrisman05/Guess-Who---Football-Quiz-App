Football Quiz Game: Guess Who?

When Start Game is clicked, a career summary of a retired or active football player will be presented. You have to guess the name of the footballer based on the summary data provided.

Rules Overview

  • You have 1 minute ⏳ to correctly guess the player's name each turn
  • Use the HINT! button if you need help making a guess
  • After 10 turns, the round will end
  • Game can be reset at any point using 🚫RESET HERE🚫

Criteria for submitting answers

  • Answers are case sensitive and space sensitive. For example:
    • ❌ alan shearer ❌ >>> ✅ Alan Shearer ✅ 
    • ❌ Alan       Shearer ❌ >>> ✅ Alan Shearer ✅
  • Accents can be omitted when submitting guesses. For example:
    • Gaël Clichy >>> ✅ Gael Clichy ✅ 
    • Jussi Jääskeläinen >>> ✅ Jussi Jaaskelainen ✅
